# NeuralNet-Zero
The goal of this research is to find a solution to the sustainable and environmental impacts of large LLMs. I will experiment with multiple different approaches and try to view if there is any effect on the performance of the model (both positive and negative) and also evaluate how the cost of training and inference is effected from these proposed changes.

### Parallel Decoding
Why do LLMs need to process one token at a time sequentially. Can this be parallelised or can multiple tokens be produced in one term to decrease the cost of inference?
